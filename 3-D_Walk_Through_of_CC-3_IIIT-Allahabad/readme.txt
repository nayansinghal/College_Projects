3-D Walk-Through of CC-3 IIIT-Allahabad

Please follow following simple steps -

1) Install the Nightly version of firefox.
   nightly.mozilla.org/

2) In nightly , type about:config in the search bar and enable - webgl.force-enabled,
   webgl.msaa-force and webgl.prefer-native-gl.
  
3) Install latest Adobe Flash Player (11.0) from http://get.adobe.com/flashplayer

4) Open index.html file.

5) Use simple mouse and keyboard operations for walkthrough.
