Universal Digital Library Management System

1. It provides the user with all the functionalities of a library.
2. The database is managed by an Administrator.
3. It also sends an automatically generated confirmation mail of book issued.
4. Index_fadeslide.html is the home page of the website.

There are some PHP files which contain SQL commands so you need to install wampserver to see all the files running in a browser.
All PHP files could not be uploaded due to memory limitations.
