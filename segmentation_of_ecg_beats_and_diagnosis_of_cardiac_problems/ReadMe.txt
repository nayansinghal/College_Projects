Here are the instructions on how to run this code.

1. Open the "system.m" file using matlab.
2. Run the source code.
3. Click on "load Data" and load the files from any of the folder - arythmia, atrial fibrillation, S-T change, normal Sinus Rhythm or supraventricular Arythmia.
4. Answer is displayed in the answer box.
5. For the segmentation of each beat, click on "Detected Beats" button.
6. For the curve fitting of beat, click on "Curve fitting" button.
